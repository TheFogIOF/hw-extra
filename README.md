### HolyWorld Extra 
HWExtra - это open source вспомогательный мод для игры на сервере HolyWorld добавляющий различные твики.

> [!Important]
> - Версия: Fabric 1.21.11
> ### Установка
> 1. Установите Fabric и Fabric API > 0.141.4 для версии Minecraft 1.21.11.
> 2. Скачайте последнюю версию мода из Releases.
> 3. Поместите .jar файл в папку mods.
> 4. Запустите игру.

### ℹ️ Краткий функционал
- Friends - система друзей, которая связывается с внешним сервером клиента для определения сервера друзей.
- Pos HUD - отображает координаты игрока. 
- Effect HUD - отображает активные эффекты игрока.
- Friends HUD - показывает список друзей.

### Внешний сервер
Сервер написан на C#, если нужно соберёте из исходников сами. В будующием за небольшую плату, все серверные функции будут доступны без поднятия своего сервера. Чтобы изменить сервер к которому обращается клиент, нужно зайти в конфиги `config\hwextra\main.json` и поменять `apiEndPoint`. 
> https://github.com/TheFogIOF/hw-extra-server

Надеюсь, что мод будет становиться лучше и лучше, свои идеи и предложения можно писать в [Issues](https://github.com/TheFogIOF/hw-extra/issues).
